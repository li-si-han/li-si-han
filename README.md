## 🔬 Research Output
### 📝 IEEE Conference Papers (Published)
1. **A Novel Approach for Integrated Optical Camera Communication and Sensing Using Reflected Light** | IWQoS 2024 | [DOI:10.1109/IWQoS61813.2024.10682865](https://doi.org/10.1109/IWQoS61813.2024.10682865)
   - Reformulated OCC as an object detection problem, proposed a novel image decoding approach by treating black and white stripes as detectable targets.
   - Optimized YOLOv8 loss function for stripe-width features, introduced dual-attention mechanism in GAN for background reconstruction.
   - Achieved >98% stripe detection accuracy, boosted object recognition from 66% to 97.5% under low SNR (-26dB to 0dB).

2. **On Simultaneous Recognition and Locating of Contextual Objects with Unmodulated Acoustic Signals Using Machine Learning** | IWQoS 2024 | [DOI:10.1109/IWQoS61813.2024.10682881](https://doi.org/10.1109/IWQoS61813.2024.10682881)
   - Addressed indoor object recognition/location by innovatively using unmodulated acoustic waves as the sensing medium.
   - Used MFCC for acoustic signal processing, developed dual-model (BiLSTM classifier + LightGBM regression).
   - 100% recognition accuracy in noise-free conditions, controlled distance estimation error within 0.5 meters.

3. **Multi-modal Channel-attention for Vehicle RGB Image Reconstruction** | ICCNC 2024 | [DOI:10.1109/ICCNC63989.2024.00018](https://doi.org/10.1109/ICCNC63989.2024.00018)
   - Built custom RGB/infrared image datasets (daytime & simulated low-light) for multi-modal fusion research.
   - Designed a multi-modal channel attention network to fuse RGB and infrared data for high-quality vehicle image reconstruction.
   - Improved SSIM from 0.65 to 0.85, MI from 0.3 to >1.5 for low-light vehicle monitoring.
